Project Complexity AI Layer
Para o SaaS de Design + Drafting Partner

Essa camada analisa automaticamente:

tipo de projeto

programa arquitetônico

tamanho da construção

número de pavimentos

features especiais

E gera um Complexity Score.

Esse score ajusta o estimate.

1️⃣ Estrutura do Complexity Score

Pontuação total:

0 – 100 points


Classificação:

0–30  → Simple Project
31–60 → Standard Project
61–80 → Complex Project
81–100 → High Complexity


Multiplicadores aplicados:

Simple → 0.90
Standard → 1.00
Complex → 1.15
High Complexity → 1.30

8️⃣ Prompt para implementar no Lovable

Use esse prompt adicional.

Create a Project Complexity AI Layer for the architectural estimate system.

The system must analyze project inputs from the form and generate a Complexity Score from 0 to 100.

Inputs analyzed:

project size
number of floors
basement
program requirements
outdoor spaces
special architectural features

Each input must add points to the complexity score.

Score classification:

0–30 → Simple
31–60 → Standard
61–80 → Complex
81–100 → High Complexity

Each category must apply a multiplier to the estimate:

Simple → 0.90
Standard → 1.00
Complex → 1.15
High Complexity → 1.30

The complexity score must update dynamically as the user fills the form.

The estimate calculation must follow this formula:

Final Estimate =
Project Size × Base Rate × Complexity Multiplier
+ Optional Services
+ Rush Fee

The system must display the complexity level in the estimate panel.

Store the following values in the admin database:

complexity_score
complexity_category
complexity_multiplier
