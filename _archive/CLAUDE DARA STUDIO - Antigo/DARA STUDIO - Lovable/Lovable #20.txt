4️⃣ Motor que decide o que mostrar

Função:

src/lib/formLogic.ts
export function shouldShow(question, answers){

if(!question.showIf) return true

const key = Object.keys(question.showIf)[0]

return answers[key] === question.showIf[key]

}

React:

{questions.map(q => {

if(!shouldShow(q, answers)) return null

return <QuestionComponent question={q} />

})}

Isso cria o formulário adaptativo automaticamente.

6️⃣ Transformar Estimate em Lead automaticamente

Quando o usuário envia o formulário:

Submit Estimate

executa:

createLead()

Arquivo:

src/lib/createLead.ts
export function createLead(data){

return {

id: Date.now(),

clientName: data.name,

email: data.email,

projectType: data.projectType,

estimate: data.estimate,

status: "New Lead",

createdAt: new Date()

}

}
