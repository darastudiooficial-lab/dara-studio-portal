Seção 01

Visão Geral e Arquitetura

Site institucional B2B para o DA•RA Studio — estúdio de suporte arquitetônico remoto de Criciúma, SC — Brasil. Single-Page Application em React 18, sem framework de roteamento externo, com formulário de orçamento de 8 etapas, portais de acesso (Cliente, Freelancer, Admin), blog técnico e sistema completo de Dark/Light Mode.

Sistema de Tema — useTheme()

Hook customizado que gerencia Dark/Light Mode com persistência em localStorage e detecção automática via prefers-color-scheme. O botão ThemeToggle (☀/🌙) aparece no nav desktop e no menu mobile.

// theme = "dark" | "light"

Seção 04

Formulário de Orçamento — 8 Steps

Coleta progressiva de dados com validação por step, cálculo de estimativa em tempo real via useMemo, envio por EmailJS e telas de sucesso com confetti. Suporta PT-BR e EN via objeto tf = T[lang].form passado como prop para cada step.

Steps e Campos

01
Localização (StepLocation)
Região (US/BR) · Endereço · Cidade · Estado · CEP/ZIP
Validação: todos obrigatórios · região define moeda ($ ou R$) e unidades (ft ou m)
02
Sobre Você (StepAboutYou)
Nome · E-mail · Telefone · Função (Builder/Architect/Developer/Owner/Other) · Empresa (nome, site, endereço)
Email com regex · telefone mín. 7 dígitos · role obrigatório
03
Detalhes do Projeto (StepDetails)
Largura × Comprimento (ft/m via FtInInput) · Pavimentos: Ground, 2nd Floor (+), Basement (+), Attic (+)
Dimensões > 0 · Input aceita 15'-2", 15-2, 15.17
04
Escopo (StepProjectScope)
13 tipos de projeto (ícones) · Engajamento: Full Construction / Floor Plans Only / PDF to CAD · Design Extras: Arch Design, Space Planning, Interior Layout, 3D Exterior, 3D Room
Mín. 1 tipo de projeto + 1 tipo de engajamento
05
Programa de Ambientes (StepProgram)
Contador +/− por categoria: Living, Bedrooms, Bathrooms, Kitchen, Office, Garage, Outdoor, Other
Opcional
06
Arquivos (StepFiles)
Upload drag-and-drop · até 10 arquivos · preview com tamanho · botão remover
Opcional — upload de ≥6 docs desbloqueia Rush e Express no step seguinte
07
Velocidade de Entrega (StepRushFees)
Standard (0%) · Rush +40% · Express +60% · Upload de 6 documentos obrigatório para desbloquear Rush/Express
Rush e Express bloqueados até upload dos documentos do projeto
08
Revisão e Envio (StepReview)
Resumo completo · Taxa estimada (faixa low-high) · Disclaimer legal · Checkbox de copyright (obrigatório) · Accept & Continue / Save for Later
Checkbox copyright obrigatório · isSending guard anti-duplo-envio · timeout 30s

Implementar Supabase Auth

// 1. Instalar
npm install @supabase/supabase-js

// 2. Criar cliente (src/supabase.js)
import { createClient } from '@supabase/supabase-js'
export const supabase = createClient(
  'https://tcyfqcfqwjqmlkvalycy.supabase.co',
  process.env.REACT_APP_SUPABASE_ANON_KEY
)

// 3. Substituir setTimeout em ClientLogin.handleSignIn
const { data, error } = await supabase.auth.signInWithPassword({ email, password })
if (error) setSiErr({ pass: error.message })
else setPage('/auth-success-client')

// 4. Google OAuth
await supabase.auth.signInWithOAuth({ provider: 'google' })
